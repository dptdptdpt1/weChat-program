# 产品概述

微信小程序项目，采用前后端分离架构。

## 技术选型

- **前端**: Taro + React + TypeScript - 使用 React 语法开发微信小程序
- **后端**: Python (FastAPI/Flask) - RESTful API 服务
- **开发理念**: 类型安全、组件化、可维护性优先

## 开发原则

- 使用 TypeScript 确保类型安全
- 组件化开发，提高代码复用性
- 前后端通过 RESTful API 通信
- 遵循微信小程序开发规范和限制
- 必要注释都使用中文注释
